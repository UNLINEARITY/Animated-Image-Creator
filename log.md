feat: 实现具有高级帧对齐编辑和日/夜模式切换的 APNG 生成器

主要改进与功能实现：

1.  **核心生成逻辑**：
    - 集成 `upng-js` 实现纯前端 APNG 编码，确保隐私安全且支持 GitHub Pages 部署。
    - 以第一张上传图片为画布基准，确保生成的动画尺寸统一。

2.  **高级编辑器 (Frame Adjustment)**：
    - 支持后续帧的平移 (Pan) 和缩放 (Zoom，最高 20x)。

    - 增加鼠标滚轮缩放支持、重置按钮以及棋盘格背景。

3.  **UI/UX 深度优化**：
    - **主题切换**：支持一键切换日间/暗黑模式。
    - **图标系统**：引入 `lucide-react`，替换所有文本按钮为专业图标。
    - **布局修正**：锁定上传区域 (Dropzone) 宽度，防止上传图片后界面跳动。
    - **细节打磨**：修复了圆形按钮内图标不居中的像素级偏差；移除 Vite 默认的居中限制，实现真正的全屏对齐。

4.  **交互与 Bug 修复**：
    - 实现帧列表的拖拽排序 (Drag & Drop sorting)。
    - 修复了 `upng-js` 编码时二进制数据格式不匹配导致文件损坏的问题。
    - 修复了 Canvas 渲染顺序导致的棋盘格遮挡图片的显示错误。
    - 完善 "Clear All" 逻辑，确保清空所有状态（包含生成的预览图）。
